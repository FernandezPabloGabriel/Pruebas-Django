# Experimentación con Django
Este repositorio está destinado para experimentar y aprender sobre el framework django mediante la realización de múltiples aplicaciones.

## App de polls
Una app que resulta de la realziación de un tutorial para construir la primera app usando Django que ofrece su documentación.
### Primera versión:
* Inicialización de primer entorno Django
* Construcción de app "polls/" que muestra un mensaje por pantalla
* Inicialización de endpoint "localhost/polls"
